# Imagens dos aplicativos

Coloque aqui as imagens (prints/telas) de cada aplicativo. O nome do arquivo deve
corresponder ao campo `imagem` em `data/catalog.json`.

Exemplos esperados pelo catalogo atual:

- `relatorio-producao.png`
- `monitor-ativos.png`
- `calculadora-logistica.png`

Formato recomendado: PNG ou JPG, proporcao ~16:9 (ex.: 1040x600).

Enquanto a imagem nao existir, a Suite exibe automaticamente um placeholder com o
emblema do app, entao nada quebra caso o arquivo esteja ausente.
